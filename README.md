# Nail Disease Prediction System

## Project Description
This project predicts nail diseases using Deep Learning (MobileNetV2 Transfer Learning).

## Technologies Used
- Python
- TensorFlow
- Flask
- HTML/CSS

## Dataset
529 training images
17 disease classes

## Model
Transfer Learning using MobileNetV2

## Accuracy
Validation Accuracy achieved: ~21%

## How to Run
1. Install requirements:
   pip install -r requirements.txt
2. Run:
   python app.py
3. Open:
   http://127.0.0.1:8080
